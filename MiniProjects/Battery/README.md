# Battery Notificator

This project is a simple battery notificator that will notify you when your battery is low. It will also notify you when your battery is fully charged.

## Pre-requisites

- Python 3.x
- plyer library

### Install Requirements

```bash
pip install plyer
```

### Usage

```bash
python battery_notificator.py
```
